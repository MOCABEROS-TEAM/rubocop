# Ruby & Rails Style Guides

### Usage
Add a git submodule
```shell
$ git submodule add https://github.com/mocaberos/rubocop.git ./vendor/rubocop
```
Inherit settings
```yaml
inherit_from: ./vendor/rubocop/rubocop.yml
```
